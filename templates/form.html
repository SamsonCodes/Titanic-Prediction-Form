Instructions

To get a prediction, fill in all fields and then press predict.

  • Make sure that your name contains some prefix like Mr., Mrs., Jonkheer. etc, as found in the Titanic data: {'Rev.', 'Mme.', 'Miss.', 'Dona.', 'Lady.', 'Sir.', 'Don.', 'Countess.', 'Jonkheer.', 'Major.', 'Col.', 'Ms.', 'Master.', 'Dr.', 'Mr.', 'Mrs.', 'Mlle.', 'Capt.'}.
  • Sex can be either [male, female].
  • Pclass can be [1, 2, 3] and refers to the class of the ticket.
  • SibSp refers to the amount of siblings / spouses aboard the Titanic. So any positive whole number will do.
  • Finally, Cabin can be any combination of a letter [A-G, T] plus number. For example C85. The number isn't used in this model though, only the letter, which refers to the Deck.

Please note that this is a rather simple model based on a limited amount of features from an already limited amount of data (891 rows). AUC ROC score (a special accuracy metric for unbalanced data ranging from 0.5-1 with 1 being perfect) of the model is 0.83, which is quite decent but not perfect. It's quite possible for the model to make wrong predictions for genuine passenger data.
